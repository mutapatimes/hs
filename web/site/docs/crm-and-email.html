CRM & email connections.

Halia writes grades and segments into the tools your team already uses, so a hidden VIC can trigger a flow the moment it's found. One thing it never does: email your customers itself. You always choose and send.

Overview

Connecting a destination lets Halia enrich your existing contacts with a grade, a score, and tags or segments you can act on. It writes to your account under a key you provide, and the same zero-retention rule applies: customers are scored in memory, and only the grade and tags you asked for are written onward.

Connect a destination from your dashboard settings. You can connect more than one at a time.

Klaviyo

KlaviyoPrivate API key
  1. Create a private API key

    In Klaviyo, go to Settings → API keys and create a Private API Key with full access to Profiles and Lists.

  2. Paste it into Halia

    In your Halia dashboard, open Integrations → Klaviyo and paste the key.

  3. Push grades

    Halia writes each customer's grade and score as profile properties, and can build a Klaviyo list or segment from any selection you make.

Segment on Halia grade or trigger a flow the moment a hidden VIC is found.

Mailchimp

MailchimpAPI key + audience
  1. Create an API key

    In Mailchimp, go to Account → Extras → API keys and generate a key.

  2. Connect and choose an audience

    Paste the key in Halia's Integrations → Mailchimp panel and pick the audience to enrich.

  3. Enrich and tag

    Halia adds grades as merge fields and applies tags such as Halia A* and Halia Hidden VIC you can segment on.

Consent is never changed. Any new contact Halia pushes is added as transactional, never subscribed — so no one is opted into marketing. Halia enriches and tags the contacts you already have.

HubSpot

HubSpotPrivate app token
  1. Create a private app

    In HubSpot, go to Settings → Integrations → Private Apps and create one named “Halia”.

  2. Grant contact scopes

    Enable crm.objects.contacts.read and crm.objects.contacts.write, then copy the access token (pat-…).

  3. Connect Halia

    Paste the token in Integrations → HubSpot. Halia creates its custom contact properties on first connect.

  4. Push graded contacts

    Halia upserts contacts by email with halia_grade, halia_score, halia_tier, halia_vic, halia_signals, halia_reasons and halia_scored_at, and can build a static list from any selection.

Slack

SlackIncoming webhook
  1. Create an incoming webhook

    In Slack, add the Incoming Webhooks app and choose the channel for alerts (for example #vic-alerts).

  2. Paste the webhook URL

    Copy the https://hooks.slack.com/… URL into Halia's Integrations → Slack panel.

  3. Get real-time alerts

    When a VIC-grade order lands, Halia posts a message with the grade and the reasons behind it — so your team can act while it matters.

Sending outreach

Halia surfaces the client and drafts the message — from a set of templates, or a personal note — but the send is always yours. Halia never emails your customers on its own. You can send from your own inbox, from WhatsApp, or by building a segment and sending natively from your ESP or Shopify Email.

This is deliberate: it keeps you in control of the relationship and keeps Halia zero-retention — there's no customer list on our side to send from in the first place.
